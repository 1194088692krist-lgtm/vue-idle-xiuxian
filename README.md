## 1. 游戏核心循环

本作是一款 **放置（idle）+ 数值成长** 修仙游戏，核心循环如下：

```
打坐修炼 / 自动修炼 → 积累修为 → 突破境界（灵力倍率×1.2 复利）
        ↓
   提升"修炼层数"(等级) → 解锁更高秘境 / 更高装备境界要求
        ↓
   挂机探索 / 抽卡 / 炼丹 → 获取装备 · 灵宠 · 素材 · 灵石
        ↓
   装备 Build + 灵宠出战 + 套装加成 + 丹药永久属性 → 战力(Build强度)飙升
        ↓
   挑战更高秘境更高难度（推荐Build匹配）→ 更优掉落 → 反哺成长
```

**最重要的一条认知**：等级（修炼层数）本身**不直接增加**攻击/生命等战斗数值。真正让你变强的是 **装备 Build、灵宠、套装、丹药永久加成、转生**。等级的作用是：
- 抬高「修为上限」与「境界名称」；
- 每次突破让 **灵力获取倍率 ×1.2 复利叠加**（越往后灵力回得越快）；
- 解锁可装备的装备「境界要求」(`requiredRealm`)；
- 作为 Build 匹配度判定外的另一条进度主线。

> 详见第 3 节「人物成长与修炼层数的关系」。

## 3. 修炼与突破：人物成长与修炼层数的关系

这是本手册最关键的一节，直接回答「人物成长和修炼层数的关系是什么？有哪些数据上面的成长」。

### 3.1 修炼层数 = 境界（共 126 层 / 14 大境 × 9 重）

`realm.js` 定义了 14 个大境界，每个 9 重，共 **126 级**：

```
练气 → 筑基 → 金丹 → 元婴 → 化神 → 返虚 → 合体 → 大乘
→ 渡劫 → 仙人 → 真仙 → 金仙 → 太乙 → 大罗
```

每一重的「修为上限 `maxCultivation`」随等级**线性阶梯式增长**（见下表）。例如练气一重 100，练气九重 900，筑基一重 1000，……大罗九重 1.4 亿。

### 3.2 打坐修炼的两条指数曲线

每次「打坐修炼」消耗灵力、获得修为，二者都随等级**指数增长**，但增速不同：

```
单次消耗灵力 = floor( 8 × 1.25^(等级-1) )
单次获得修为 = floor( 3 × 1.28^(等级-1) )
```

因为 **修为增速(1.28) > 消耗增速(1.25)**，且修为上限只是线性增长，所以：
- **低等级**：需要几十次打坐才能填满一重；
- **高等级**：一次打坐获得的修为远超当前上限，会**自动连跨多阶突破**（越到后期进度越快，不会卡死）。

### 3.4 突破（tryBreakthrough）带来的数据成长

每次突破成功，发生以下**永久性**变化：

```
灵力     += 100 × 当前等级
灵力倍率  spiritRate ×= 1.2      ← 复利！每次突破灵力回得更快
境界名称  更新为下一重
修为      清零，修为上限改为下一重的 maxCultivation
突破次数  +1
```

**突破成功率**：
```
baseChance = max(0.6, 1 - (等级-1) × 0.015)
最终成功率  = min(1, baseChance + 渡厄丹加成)
```
- 1 级 100%，约 27 级后触底 **60%** 地板；
- 高境界频繁失败，建议用 **渡厄丹**（`breakthroughRate +10%`，可叠加至 +90%）兜底。

### 3.5 被动修为（离线/挂机也涨）

`regenerateSpirit()` 在恢复灵力时，按在线真实时间给被动修为：
```
passiveCultRate = max(1, floor(等级×0.8 + Build强度×0.00001))
passiveGain     = floor(经过毫秒 / 10000) × passiveCultRate
```
- 等级越高被动越快；Build 越强也有微幅加成；
- 相当于「放着不动也涨修为」，是放置玩法的核心福利。

### 3.6 结论：成长到底从哪来？

| 维度 | 由等级直接决定？ | 实际来源 |
| --- | --- | --- |
| 修为上限 / 境界名 | ✅ 是 | realm.js 线性阶梯 |
| 灵力回复速度 | ✅ 是（突破复利 ×1.2） | 突破次数 |
| 突破成功率 | ✅ 是（线性衰减至 60%） | 等级 |
| 攻击 / 生命 / 防御 / 速度 | ❌ 否 | 装备、灵宠、套装、丹药永久加成、转生 |
| 暴击/连击/闪避等战斗属性 | ❌ 否 | 装备词条、套装、灵宠 |
| 可装备的装备 | ⚠️ 间接（requiredRealm 门槛） | 等级解锁高境装备 → 抬高 Build |

> **一句话总结**：修炼层数决定「你能走多高、灵力回多快、能穿什么装备」；而「你有多强」取决于 Build（装备+灵宠+套装+丹药+转生）。两者通过「装备境界要求」衔接——等级不够，再好的神品也穿不上。

## 4. 转生渡劫系统

达到 **50 级** 可转生，重置等级为 1 但获得**永久属性加成**：

```
第 N 世转生加成：
  攻击 +5N    生命 +50N    防御 +3N    速度 +2N
  修炼效率 +10%N    灵力效率 +10%N
```
- 加成**永久沉淀**到 `baseAttributes` 与倍率，越转越强；
- 灵石、装备、物品**全部保留**；
- 适合中后期卡 Build 或想冲极限数值的玩家。

## 5. 装备与 Build 体系

### 5.1 装备品质（rarity）

| 品质 | 名称 | 颜色 | 评分倍率 | 词条数 |
| --- | --- | --- | --- | --- |
| common | 凡品 | 灰 | ×1 | 0 |
| uncommon | 良品 | 绿 | ×1.5 | 0~1 |
| rare | 上品 | 蓝 | ×2 | 1~2 |
| epic | 极品 | 紫 | ×3.5 | 2~3 |
| legendary | 仙品 | 金 | ×6 | 3~4 |
| mythic | 神品 | 红 | ×10 | 4~5 |

### 5.2 装备槽位（13 个）

```
weapon 武器 / head 头部 / body 衣服 / legs 裤子 / feet 鞋子
shoulder 肩甲 / hands 手套 / wrist 护腕 / necklace 项链
ring1 戒指 / ring2 戒指 / belt 腰带 / artifact 法宝
```
> 法宝 `artifact` 仅由高难 Boss / 抽卡产出，是 Build 的强力补强点。

### 5.3 词缀（affix）与品质倍率

- 词条按 `tier` 分档：`tier1 ×1`、`tier2 ×1.5`、`tier3 ×2.5`；
- 高 tier 词条（如 `增伤/减伤/战意/抗性`）对 Build 评分贡献更大；
- 单件装备评分公式：
```
装备评分 = round( (基础分 + 词条分 + 套装分) × 品质倍率 × (1 + 强化等级×0.1) )
```
  - 基础分来自 `attack/health/defense/speed` 等（`attack` 5 分/点、生命 0.5、防御 3、速度 8）；
  - 战斗类百分比属性 300 分/点、抗性类 250 分/点、灵力/修炼类 100 分/点；
  - 带套装 `setId` 固定 +50 分。

### 5.4 套装（setBonuses）

集齐同套装多件触发百分比加成，套装分额外计入 Build：

| 套装件数 | Build 加成分 |
| --- | --- |
| 2 件 | +200 |
| 3 件 | +300 |
| 4 件 | +500 |
| 5 件 | +800 |

游戏内置 10 套（烈焰刃/玄铁卫/疾风影/血牙/雷霆击/不灭体/反击道/灵气髓/战斗王/道仙尊），覆盖攻、防、速、吸血、暴击、生命、修炼等方向。集齐 4~5 件套是 Build 飙升的关键。

### 5.5 Build 强度（总评分）

```
Build强度 = Σ(每件已装备评分) + Σ(套装激活分)
```
- 在「修炼/人物」页可见 `Build强度` 数值；
- 这是挂机「推荐 Build 匹配度」的判定基准。

### 5.6 Build 强度参考阶梯（BUILD_TIERS）

| 档位 | 名称 | 参考 Build | 含义 |
| --- | --- | --- | --- |
| 入门 | 入门 | 68,000 | 全稀有(+10)基础满装 |
| 强力 | 强力 | 686,000 | 传说/史诗混搭(+20)含套装 |
| 顶级 | 顶级 | 5,090,000 | 全神话(+50)含套装 |
| 极限 | 极限 | 20,000,000 | 全神话(+100)双五件套·混沌级 |

> 用这套阶梯对照自己当前 Build，能直观判断「够不够去某张图打某难度」。

### 5.7 装备强化 / 洗练 / 出售

- **强化**：每级 `评分 ×(1+0.1×强化等级)`，吃强化石；可用**淬灵丹**提升下次成功率；
- **洗练**：重roll词条，吃洗练石；可用**定灵丹**保底不降属性；
- **出售**：`max(1, round(评分 × 0.1))` 灵石（一键出售低品质回血好用）；
- **分解**：产出强化石/洗练石（见第 2 节表）。

## 6. 灵宠系统

灵宠出战后为人物提供**全属性加成**（攻击/防御/生命 + 一众战斗属性）。加成由「品质 × 星级 × 等级 × 阶位」综合决定：

```
qualityBonusMap:  divine 15% / celestial 12% / mystic 9% / spiritual 6% / mortal 3%
每星加成:         各品质 1%~2%/星
每级加成:         (等级-1) × 基础加成 × 10%
每 5 星阶位:      阶位 × 基础加成 × 50%
最终加成 = 基础 + 星 + 等级 + 阶位（战斗属性取一半）
```

- **升级**（`upgradePet`）：吃灵宠精华，按品质倍率（divine 2.0 / celestial 1.8 / mystic 1.6 / spiritual 1.4 / mortal 1.2）提升战斗属性；
- **升星**（`evolvePet`）：消耗同名同品质灵宠，星级 +1，并返还被吃灵宠已耗精华；
- **出战/召回**：同一时刻仅 1 只出战，加成实时叠加到人物属性。

> ⚠️ 重要：灵宠物体必须带 `combatAttributes` 字段（攻击/防御/生命/各战斗属性）才能被正确读取加成。挂机产出的灵宠现已对齐此结构，**出战即可生效**。

### 7.2 单件属性强度（挂机生成，已上调）

```
装备:  攻击 = floor(区推荐攻击 × 0.22 × 品质倍率)
       生命 = floor(区推荐生命 × 0.16 × 品质倍率)
       防御 = floor(区推荐攻击 × 0.13 × 品质倍率)
       速度 = floor(8 × 品质倍率)
灵宠:  攻击 = floor(区推荐攻击 × 0.16 × 灵宠品质倍率)
       防御 = floor(区推荐攻击 × 0.09 × 灵宠品质倍率)
       生命 = floor(区推荐生命 × 0.09 × 灵宠品质倍率)
       其余战斗属性按品质倍率给到 4%~50% 上限
```
品质倍率——装备：`common1 / uncommon1.3 / rare1.8 / epic2.5 / legendary4 / mythic7`；灵宠：`mortal1 / spiritual1.5 / mystic2 / celestial3 / divine5`。

示例（龙渊，推荐攻击100/生命800，出极品 epic 装备）：攻击≈55、生命≈320、防御≈32、速度≈20；出玄品 mystic 灵宠：攻击≈32、生命≈144、防御≈18。

### 7.3 推荐 Build 与匹配度（挂机成败核心）

每张图有「凶险档基础推荐 Build」(`ZONE_BUILD_BASE`)，再按难度缩放：

```
推荐Build(某难度) = 基础Build × 难度缩放系数
匹配度 = 你的Build强度 ÷ 推荐Build
```
- 匹配度 ≥ 100%：稳定挂机，气血稳步回升；
- 匹配度 < 100%：按不足比例持续掉血，**气血归零即力竭提前结束**；
- 胜利额外回血（含少量擦伤），失败重创（Build 越弱伤越重）。

| --- | --- | ---: | ---: | ---: | ---: |

## 10. 丹药炼制系统

丹药按「品阶 grade (1~9)」决定**基础成功率**（一品 90% → 九品 10%）与材料需求（残页 = `5×品阶+5`）。按功能分四类，效果随等级 `×(1+(等级-1)×10%)` 放大：

| 类型 | 倍率 | 代表丹药 |
| --- | --- | --- |
| 灵力类 | ×1.0 | 聚灵丹、回灵丹、日月丹(灵力上限) |
| 修炼类 | ×1.2 | 聚气丹、凝元丹、天元丹 |
| 属性类 | ×1.5 | 雷灵丹、五行丹、火元丹 |
| 特殊类 | ×2.0 | 仙灵丹、涅槃丹、清心丹、渡厄丹 |

**功能细分（关键丹药）**：

| 丹药 | 功能 | 效果 |
| --- | --- | --- |
| 洗髓丹 | 永久属性 | 永久 +5 攻击（存档沉淀） |
| 锻骨丹 | 永久属性 | 永久 +6 防御 |

## 11. 战斗系统公式

战斗为**回合制**（最多 10 回合，超时判负），按速度决定先手。伤害结算顺序：

```
1) 基础伤害 = 攻击 × (1 + 战斗属性提升 combatBoost)
2) 暴击：最终暴击率 = 攻方暴击率×(1+combatBoost) − 守方抗暴×(1+抗性提升)
         触发 → 伤害 ×= (1.5 + 暴伤强化)
3) 连击：触发 → 伤害 ×= 1.3
4) 吸血：触发 → 回血 = 造成伤害 × 30%
5) 眩晕：触发 → 守方本回合无法反击
6) 最终增伤：伤害 ×= (1 + 最终增伤)
7) 防御减伤：伤害 ×= 100 / (100 + 有效防御)，有效防御 = 防御×(1+combatBoost)
8) 若本次为暴击 → 再 ×= (1 − 暴伤减免)
9) 最终减伤：伤害 ×= (1 − 最终减伤)
```
- **闪避**：实际闪避率 = 守方闪避率 − 攻方抗闪避；命中则免伤；
- **反击**：守方反击率 − 攻方抗反击；触发则反打；
- **先手**：速度高者先攻（含 combatBoost 加成）。

> 实战要点：**最终增伤/减伤、暴伤、吸血、连击** 是后期碾压的核心；堆叠 `combatBoost` 与 `resistanceBoost` 能同时放大攻防两端。

## 12. 进阶策略与成长路线

### 前期（练气 ~ 金丹，1~27 级）
- 主线：手动/自动打坐把修为堆上去，靠突破把 `spiritRate` 复利拉起来；
- 资源：低难挂机（青萝林/迷雾谷 游历~试炼）攒灵石、灵草；
- 炼丹：优先 **聚灵丹(灵力)、聚气丹(修炼)、洗髓/锻骨丹(永久属性)**；
- 装备：抽卡综合池拿第一套良品/上品，凑 2 件套即可。

### 中期（元婴 ~ 化神，28~45 级）
- 目标：Build 冲到 **入门(6.8万)~强力(68.6万)**；
- 装备：进凤凰窟/龙渊 凶险~绝境，挂机出极品/仙品；集齐 1~2 套 4 件套；
- 灵宠：玄品/仙品出战，开始升级+升星；
- 突破：60% 地板后用 **渡厄丹** 兜底，避免卡境界；
- 丹药：悟道丹+寻宝丹叠加，挂机收益最大化。

### 后期（返虚 ~ 大乘，46~72 级）
- 目标：Build 冲 **顶级(509万)**，挑战鬼荒原/冰雪宫 绝境~灭世；
- 装备：神品为主，法宝(artifact)优先补位；强化 +20~+50；
- 灵宠：仙品/神品满星，精华集中养 1~2 只核心；
- 转生：满 50 级可转生，永久 +攻/血/防/速 与 10% 效率，越转越强。

### 终局（渡劫 ~ 大罗，73~126 级）
- 目标：Build 逼近 **极限(2000万)**，混沌界 灭世稳定挂机；
- 全神话双五件套、法宝满配、神品满星灵宠、多次转生叠永久属性；
- 利用被动修为与离线结算，放置即可持续逼近大罗九重 1.4 亿修为上限。

### 通用技巧
1. **Build 优先于等级**：与其猛冲等级，不如先把 Build 堆到匹配当前图，挂机效率最高；
2. **套装 > 散件**：4~5 件套的 Build 分与百分比加成远超单件；
3. **灵力倍率复利**是隐藏引擎：多突破 = 灵力回得快 = 修炼不停；
4. **永久丹药**早吃早享受，转生不丢；
5. **匹配度 < 100% 别硬挂**：会力竭提前结束，先把 Build 或难度降一档。

## 13. 数据附录：关键公式汇总

```js
// —— 修炼 ——
单次消耗灵力 = floor(8 × 1.25^(等级-1))
单次获得修为 = floor(3 × 1.28^(等级-1))
突破成功率   = min(1, max(0.6, 1 - (等级-1)×0.015) + 渡厄丹)
突破奖励     = 灵力 +100×等级;  spiritRate ×= 1.2
被动修为/10s = max(1, floor(等级×0.8 + Build×0.00001))
修为上限     = realm.js 线性阶梯（100 → 1.4亿）

// —— 装备评分 ——
装备评分 = round((基础分 + 词条分 + 套装分) × 品质倍率 × (1 + 强化等级×0.1))
品质倍率   = common1 / uncommon1.5 / rare2 / epic3.5 / legendary6 / mythic10
词条tier倍 = tier1×1 / tier2×1.5 / tier3×2.5
套装激活分 = 2件+200 / 3件+300 / 4件+500 / 5件+800
Build强度  = Σ(已装备评分) + Σ(套装激活分)

// —— 挂机 ——
推荐Build(难度) = ZONE_BUILD_BASE × 难度缩放(0.3/0.6/1.0/1.6/2.5)
匹配度         = 你的Build ÷ 推荐Build
胜利修为       = round(5 × 地图难度 × (1 + 匹配度×0.5))
挂机装备属性   = 区推荐×(攻击0.22 / 生命0.16 / 防御0.13 / 速度8) × 品质倍率
挂机灵宠属性   = 区推荐×(攻击0.16 / 防御0.09 / 生命0.09) × 灵宠品质倍率

// —— 灵宠加成 ——
最终加成 = 品质基础(3%~15%) + 星级(1%~2%/星) + 等级(基础×10%×(级-1)) + 阶位(基础×50%×floor(星/5))
战斗属性取最终加成的一半

// —— 战斗 ——
伤害 = 攻击×(1+combatBoost) ×[暴击1.5+暴伤] ×[连击1.3] ×(1+最终增伤)
     × 100/(100+防御×(1+combatBoost)) ×(1−暴伤减免,若暴击) ×(1−最终减伤)
闪避 = 守方闪避 − 攻方抗闪避；  反击 = 守方反击 − 攻方抗反击
```
