# 修仙问道 — 50 角色动态立绘 MP4 生成提示词

## 全局技术规格

```
任务：将给定的静态角色立绘 PNG 转化为 3-5 秒循环动画 MP4 视频

=== 输出格式 ===
- 编码：H.264 MP4
- 分辨率：480×720 像素（竖屏 2:3 比例）
- 帧率：24fps
- 时长：4 秒（96 帧）
- 音轨：无（静音视频）
- 文件大小：单个不超过 500KB

=== 首尾帧硬性要求（关键！） ===
- 第 0 帧（首帧）必须与输入的静态立绘 PNG 完全一致（逐像素匹配）
- 最后一帧（第 95 帧）也必须与第 0 帧完全一致
- 这样视频循环播放时，最后一帧→第一帧 无缝衔接，无跳变
- 实现方式：所有动效使用正弦函数（sin/cos）驱动，确保周期 = 视频时长，首尾自然归位
- 动效从第 1 帧开始缓慢启动，在第 94 帧缓慢停止，首尾帧保持静止原图

=== 动画风格规范 ===
- 动效幅度：微小、自然，不要大幅度动作（放置类游戏，立绘应安静优雅）
- 循环：首尾帧完全一致，确保无限循环无跳变
- 光源：保持与原图一致（左上方 45° 主光源）
- 面部表情：不改变（保持原图表情）
- 构图位置：角色保持在画面同一位置，不偏移
- 色调：保持原图色调不变
- 禁止添加任何文字、水印、边框

=== 动效元素库（每个角色选取 2-4 种组合） ===
A. 呼吸：身体以正弦函数上下起伏（幅度 2-4px），周期 = 视频时长
B. 前发飘动：刘海/前发以正弦函数左右摆动（幅度 2-3px），与呼吸不同相位
C. 后发飘动：后发以正弦函数摆动（幅度 1-2px），与前发有延迟
D. 衣袂飘动：衣摆/袖口/飘带以正弦函数飘动（幅度 2-4px）
E. 粒子特效：属性对应粒子以正弦轨迹运动，粒子在首尾帧不可见或回到原位
F. 光晕呼吸：光环/光晕以正弦函数明暗变化，周期 = 视频时长
G. 悬浮物浮动：悬浮道具/武器以正弦函数上下浮动（独立于身体）

=== 粒子运动规则（确保首尾帧一致） ===
- 粒子运动轨迹必须是闭合回路（圆形/椭圆形/正弦往返）
- 粒子在首帧和尾帧的位置、透明度必须完全相同
- 严禁粒子从画面外飘入再飘出（会导致循环跳变）
- 正确做法：粒子在固定范围内做圆周运动或往复运动

=== 文件命名与输出 ===
输入：char_XXX_name.png（静态立绘）
输出：char_XXX_name.mp4（动画视频）
输出目录：portraits/videos/
```

## 技术实现要点（给 Agent 的数学参考）

```
所有动效必须使用正弦函数驱动，确保首尾帧自然归位：

// 示例：4 秒视频（96 帧），24fps
const totalFrames = 96;
const duration = 4; // 秒

// 呼吸动效（首尾帧值为 0）
function breathe(frame) {
  const t = frame / totalFrames;          // 0 ~ 1
  return Math.sin(t * Math.PI * 2) * 3;   // 幅度 3px，周期 = 视频时长
}
// frame=0: sin(0)=0, frame=48: sin(π)=0, frame=95: sin(2π*95/96)≈0 ✓

// 头发飘动（相位偏移）
function hairSway(frame) {
  const t = frame / totalFrames;
  return Math.sin(t * Math.PI * 2 + 0.5) * 2; // 相位偏移 0.5
}
// 注意：相位偏移不影响首尾帧归零，因为 sin 周期仍 = 视频时长

// 粒子圆周运动（首尾帧位置重合）
function particleCircle(frame, radius, speed) {
  const t = frame / totalFrames;
  const angle = t * Math.PI * 2 * speed; // speed=1 时一周归位
  return {
    x: Math.cos(angle) * radius,
    y: Math.sin(angle) * radius
  };
}
// speed 必须为整数（1, 2, 3...），确保首尾帧位置完全重合

// 光晕明暗（首尾帧亮度一致）
function glowBreath(frame) {
  const t = frame / totalFrames;
  return 0.8 + Math.sin(t * Math.PI * 2) * 0.2; // 0.6 ~ 1.0 ~ 0.8
}
// frame=0: 0.8, frame=48: 0.8, frame=95: ≈0.8 ✓
```
